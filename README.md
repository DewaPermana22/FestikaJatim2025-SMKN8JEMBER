
# AutoFile Manager

_Aplikasi Console Python untuk Otomasi & Manajemen File_

AutoFile Manager adalah aplikasi berbasis console yang dibuat menggunakan Python, dirancang untuk memudahkan user dalam mengelola file secara cepat, praktis, dan efisien. Aplikasi ini menyediakan fitur seperti sinkronisasi folder, backup, pembersihan file lama, hingga rename massal, semuanya dapat dijalankan langsung melalui terminal.
